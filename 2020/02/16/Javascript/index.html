자바스크립트 공부.

Created at 2020-02-16 Updated at 2020-02-19

자바 스크립트 공부.

const = 상수, 한번 정하면 바꾸지 못함.
null = 없다.
undefined = 정해지지않았다.

-연산자
산술 연산자.
사칙 연산같은것을 수행하는 연산자.
대입 연산자.
let a = 1;
a += 3;
console.log(a); = 4 이런식으로 대입해주는거.
논리 연산자.(쓴 순서대로 우선 실행된다.)
NOT !
AND && 양쪽의 값이 모두 true일때만 true이다.
OR || 둘중에 하나라도 true면 true.
-비교연산자
=== 세번 입력. 같다는 뜻이 아니라 비교하는것임.
=,<,> 보통 기호와 같음. 대신 <= 이런식으로 이퀄이 뒤로와야함.
문자열 붙이기
const a = ‘안녕’
const b = ‘하세요’
console.log(a + b); = 안녕 하세요.

-조건문.
if : 만약 이 조건이 참이면 실행됨.
else : if나 else if 조건이 거짓이여서 실행안되면 else 실행됨.
else if :여러개 입력 가능. 조건이 참이면 실행.
switch case : 특정한값이 될시 실행되게함.
중간에 break해줘야하고 마지막 그 조건이 아닐시 실행은 default.

-함수.
특정 코드를 하나의 명령어로 실행하게해주는 기능.Resturn 반환시 함수 끝남.
Template literal :

화살표 함수 : => 값 쓰면 Return 안써줘도 Return 됨.

-객체.
어떠한 값을 선언할때 하나의 이름으로 여러값을 넣어주게함.

비구조화 할당.
특정 값들을 객체 밖으로 빼내서 코드가 깔끔하게 만듦.

객체 안에 함수 넣기.
함수안에있는 this는 자기안에 속해있는것만 가리킴.
